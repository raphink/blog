I am just testing Ubuntu Linux for PPC as I've seen it seems to be a good new distrib.  
  
Well I'll begin with good points :  

-   it is based on Debian => rock solid
-   it installed without any pb on my parents' eMac, and this is wow! This is the first distrib of linux PPC that I manage to install on this computer without having to hack anything around...
-   the installation process is very well done and automatic
-   my internal material was all recognized automatically. Well it's not exactly the case of my printer and scanner though...
  

  
  
Bad points now :  

-   where is KDE?? This distrib comes with Gnome only smiley('cry'); and I couldn't even install it with the universe non-official packages or the KDE cvs!
-   there are still very few packages available (not even talking of XWindows ...)
-   after 2 hours, the graphical package manager crashed, so I have to use apt-get in a console
-   I don't like the default graphics ... can't help it

  
  
I think this is a good starting point for a new distrib (esp. as I tested it on a G4 PPC), but it still has quite a lot to go before I can say I prefer it to Mandrake ... Let us hope Kubuntu (Ubuntu with native KDE project) will develop soon smiley('grin2');  
  
For more infos about Ubuntu Linux, see [the official Ubuntu Linux website](http://web.archive.org/web/20050308232548/http://www.ubuntulinux.org/).
